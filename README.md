# Stress Detection and Management using Machine Learning and Image Processing

## Project Overview
This project detects stress levels using facial expressions through image processing and machine learning techniques.

## Technologies Used
- Python
- OpenCV
- Machine Learning
- NumPy
- Scikit-learn

## Features
- Real-time face detection
- Stress level prediction
- Webcam-based monitoring

## Installation
1. Clone the repository
2. Install requirements

pip install -r requirements.txt

## Run the project

python main.py
